1.onclick事件绑定

我们在浏览网页时，会通过鼠标点击来进行一些操作。要实现这些操作我们需要用到onclick事件绑定

<!--在div标签里，给div添加一个onclick事件-->

<div onclick = alert(“点我！点我！！点到我了！！！ "); ></div>

2.元素获取

document.getElementById(); 通过标签的id值来获取标签

document.getElementsByClassName(); 通过标签的class值来获取标签（返回的是一个数组，将所有相同class名的标签都返回）相同的ClassName可以通过下标来区分

document.getElementsByName(); //通过标签的名字来获取标签（返回的是一个数组，将所有相同的标签名的标签返回）相同的name可以通过下标来区分

document.getElementsByTagName(); // 通过标签的name值来获取标签 （返回的是一个数组，将所有相同的name值的标签返回）相同的TagName可以通过下标来区分

3.变量

通常我们使用var关键字来声明变量

使用关键字var+变量名=值；就可以定义一个变量。

示例：定义一个变量：var a=12; 以上代码定义了一个变量a，变量里保存的值是数值12

变量名的命名规范

  • 1.使用字母，下划线，数字，美元符组成
  • 2.首位不能是数字
  • 3.多个单词使用驼峰算法（即单词首字母大写）
  • 4.不能使用关键字和保留字
  • 5.尽量使用有语义的单词
  • 6.不能有空格

4.数据类型

4.1 数值类型 number

number是数值类型,在生活中,我们把没有小数位的数称作整数,带有小数位的数称 作小数(计算机中叫浮点数),JavaScript不区分整数和浮点数,统一用Number表示。

var a=123;整数123

var a = 0.456;浮点数0.456

var a = 1.2345e3;科学计数法表示1.2345x1000 等同于1234.5

var a = -99; 负数

NaN; NaN表示 Not a Number，当无法计算结果是用 NaN表示

Infinity; Infinity表示无限大，当数值超过 JavaScript的 Number 所能表示的最大值时，就表示Infinity

4.2 字符串 string

字符串是以单引号’或双引号”括起来的任意文本,比如 ‘abc’ , “xyz” 等等。请注意, ‘’ 或 “” 本身只是一种表示方式,不是字符串的一部分,因此,字符串 ‘abc’ 只有a, b,c这3个字符.

字符串只能用“”或‘’来包裹，如果字符里面含有“”或‘’就只能去与之不同的符号来包裹。如果字符串内部既包含了“”又包含了‘可以用转义字符\来标识
比如'I'm\"ok\"!;表示字符串内容是：I’m”ok”!

4.3 布尔值 boolean

在JS中，我们使用true来表示真，用false来表示假。布尔值只有真核假，没有第三个值。通常用在条件判断中。

4.4 null和undefined

null 表示一个“空”的值，它和0以及空字符串“”不同，0是一个数值，“”表示长度为0的字符串，而null表示“空”。

undefined，表示“未定义”。

大多数情况下,我们都应该用null。 undefined仅仅在判断函数参数是否传递的情况下有用。
